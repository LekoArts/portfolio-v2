## Editor & Terminal

- I primarily use my desktop PC with Windows 10 and occasionally my MacBook Pro with MacOS
- [Webstorm](https://www.jetbrains.com/webstorm/) & [Visual Studio Code](https://code.visualstudio.com/)
- [JetBrains Mono](https://www.jetbrains.com/lp/mono/) as preferred font
- [zsh](https://www.zsh.org/) + [oh-my-zsh](https://ohmyz.sh/)
  - [spaceship](https://github.com/denysdovhan/spaceship-prompt) as zsh prompt
  - Plugins: git, zsh-autosuggestions, virtualenv, yarn, zsh-syntax-highlighting, history, gh
- [Windows Terminal](https://github.com/microsoft/terminal) & [iTerm2](https://iterm2.com/)
- [nvm](https://github.com/nvm-sh/nvm) as Node.js version manager
- On Windows 10 I use [WSL (Ubuntu)](https://ubuntu.com/wsl) as package manager for developing, on MacOS I use [MacPorts](https://www.macports.org/)

## Software

- [Adobe Photoshop](https://www.adobe.com/products/photoshop.html) & [CaptureOne](https://www.captureone.com) for photo editing
- [Cinema4D](https://www.maxon.net/en/cinema-4d/) for 3D art + [Arnold](https://www.arnoldrenderer.com/) as renderer
- [Figma](https://www.figma.com/) for designs
- [Duplicati](https://www.duplicati.com/) for offline + online backup strategy
- [Enpass](https://www.enpass.io/) as password manager
- [f.lux](https://justgetflux.com/) for proper coding night sessions
- [FastStone Photo Resizer](https://www.faststone.org/FSResizerDetail.htm) as image batch processing tool
- [ShareX](https://getsharex.com/) for taking screenshots, recording quick videos/gifs
- [Streamlabs OBS](https://streamlabs.com/) for recording videos
